Create repository on GitHub

  • Enter a unique name for your repository
  • Choose either public or private(private will be only visible to you)
  • Then create repository

Initialize Git

  • Then using cmd go to the directory where you want to initialize git local repository using git init
  • This will make a .git folder in the current directory

Create file

  • In current folder make a file(.html here)
  • And save

Git Status

  • You can use git status to check which files are up to date in your local repository
  • Files in red are those which are not up to date while files in green are

Add file(s)

  • After making a file use git add "filename" to add file to local git repository
  • You can use git add . to add all the files/folders in current folder to local repository
  • This will only add the files which have been edited or are new

Git Commit

  • After adding files to local repository you can use git commit -m "message"to commit the files locally
  • The "message here usually describes what changes have been made in this commit"

Add origin

  • To add origin use git remote add origin [URL]
  • The git remote command lets you create, view, and delete connections to other repositories.
  • The git remote -v command lists the URLs of the remote connections you have to other repositories.

Git Push

  • To push the code to remote repository use git push origin [branch name]
  • After git push origin main terminal will get your github username and password
  • And now if we go and check our repository page on GitHub it will have all the files or folders that we committed to our local repository
